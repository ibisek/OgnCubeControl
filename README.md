# Ogn Cube Control

Android app to control OGN Cube tracker. 

It shall help you with
* updating firmware,
* maintaining your log book,
* examining tracks of the flights,
* configuration of the tracker unit,
* relationship issues ;)

Firmware flashing is the first working feature! :)
